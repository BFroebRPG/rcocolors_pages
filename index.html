Functions Included

This package includes four functions:

-rco_cols() -rco_color_pal() -scale_color_rco() -scale_fill_rco()

rco_cols

takes a input string or group of strings and returns the hex code. The color must be one of the colors included in this package

Example:
rco_cols("blue1")
##     blue1 
## "#2C3B97"
rco_cols("teal1", "brown3")
##     teal1    brown3 
## "#40BEB1" "#802F19"

rco_color_pal

Converts an input string representing a color palette into a function that can be converted into a usable color palette.

rco_color_pal("blue to orange")
## function (n) 
## {
##     x <- ramp(seq.int(0, 1, length.out = n))
##     if (ncol(x) == 4L) 
##         rgb(x[, 1L], x[, 2L], x[, 3L], x[, 4L], maxColorValue = 255)
##     else rgb(x[, 1L], x[, 2L], x[, 3L], maxColorValue = 255)
## }
## <bytecode: 0x000001e4b40ef4f0>
## <environment: 0x000001e4ca0ff1f8>

By adding a step count after the function it will return a usable palette

rco_color_pal("blue to orange")(10)
##  [1] "#2C3B97" "#33429B" "#3A4A9F" "#57747F" "#7BAA54" "#95AA39" "#A6722E" "#B94925" "#D54F23" "#F15622"

These palettes can be reversed as needed

rco_color_pal("blue to orange", reverse = TRUE)(10)
##  [1] "#F15622" "#D54F23" "#B94925" "#A6712E" "#95AA39" "#7BAA54" "#57747F" "#3A4A9F" "#33429B" "#2C3B97"

scale_color_rco and scale_fill_rco

These functions supply a fill or color to a ggplot call.
